ubuntu16.04 apt-get install lxde error

error 1 can't find lxde menu item on lightdm

apt-get install lxde and reboot can't find lxde menu item in lightdm .

apt-get install lxsession

elwin@elwin-virtual-machine:~$ ls /usr/share/xsessions/

openbox.desktop ubuntu.desktop

find no lxde related *.desktop file.

we find /usr/share/xsessions/LXDE.desktop on ubuntu 14.04 in lxsession package. but on ubuntu 16.04 apt-get isntall lxde does not install lxsession package.

we shoule apt-get install lxsession reboot system we can find lxde menu item on lightdm, and login lxde desktop envirionment.

error 2 can't logout lxde session by clicking on logout menu item.

after login lxde desktop environment ,can not logout by click on logout menu item. no error report , no any response.

after strace the lxpanel process ,we find the error messge :

"/usr/bin/lxde-logout: line 2: lxsession-logout: command not found\n"

there is no lxsession-logout program, we should apt-get install lxsession-logout

then we can logout lxde session.
